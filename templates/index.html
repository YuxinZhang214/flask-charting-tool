Welcome to Charting Tool

This tool helps you generate high-quality visualizations based on your given CSV file.

Supported Charts

  • Scatter Plot
  • Bubble Charts
  • Bar Charts
  • Line Charts
  • Pie Charts
  • Contour Graph

How to Use

To use this tool, follow these steps:

  1. Upload your CSV file
  2. Select the type of chart you want to generate
  3. Customize your chart as needed
  4. Download your generated chart
